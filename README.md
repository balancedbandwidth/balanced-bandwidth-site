# Balanced Bandwidth Next.js Starter

Starter Next.js App Router site for Balanced Bandwidth, focused on Fractional IT Leadership.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Notes

- Original uploaded HTML is preserved as `original-homepage.html`.
- Contact form markup is Netlify Forms-ready. Update if deploying to Vercel or another form handler.
- Add your final logo to `/public` and replace the text logo in `components/Header.tsx` and `components/Footer.tsx` if desired.
